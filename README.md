# ToastMenu-Lite-Typing-Club-
This is a bookmarklet for my Typing Club menu. (Limited Features, see Tampermonkey version for the full release.)

# Features:

Menu
-Draggable Menu
-Dropdown buttons

Auto Typing
-Auto Type (Human speed)
-Fast Type (Types at an average of 100 WPM)
-Slow Type (Types at an average of 40 WPM)
-Smart Type (Auto types until 95% of the lesson is complete, then requires manual typing)

Lesson
-Copy Lesson Text (Copies lesson text to clipboard)
-Skip Lesson (Skips the current lesson)

Apperance
-Background Color (Changes background to the desired color. Uses a hex value or the name of a color.)
-Background Image (Changes the background to the desired image or gif. It must be a link!)
-Text Color (Changes the color of the text to the desired color. Uses a hex value or the name of a color.)

Utilities
-NW! Rename to ToastMenu User (Sets name to ToastMenu User)
